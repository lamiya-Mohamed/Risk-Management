# نظام إدارة المخاطر (Risk Management System)

هذا المشروع هو **تطبيق ويب لإدارة المخاطر** باستخدام لغة البرمجة Python ومكتبة Streamlit.  
يتيح التطبيق إضافة المخاطر، عرضها، البحث عنها، تحديثها، حذفها، وعرض ملخص كامل لكل المخاطر المسجلة.

---

## مميزات التطبيق

- إضافة خطر جديد مع تحديد:  
  - اسم الخطر  
  - وصف الخطر  
  - نوع الخطر (تشغيلية، تكنولوجية، إلخ)  
  - درجة الاحتمالية (من 1 إلى 5)  
  - درجة التأثير (من 1 إلى 5)  
  - حساب مستوى الخطر تلقائيًا
- عرض قائمة المخاطر المسجلة
- البحث عن المخاطر بالاسم أو النوع
- تحديث بيانات أي خطر موجود
- حذف أي خطر
- عرض ملخص المخاطر:  
  - إجمالي عدد المخاطر  
  - متوسط مستوى المخاطر  
  - عدد المخاطر العالية
- حفظ جميع المخاطر في ملف JSON
